# 🏥 SurgiTrack AI

## RFID-Based Smart Surgical Instrument Tracking System

SurgiTrack AI is a Biomedical Engineering project developed to improve
Operating Theatre (OT) instrument management using RFID technology,
ESP32, Flask, and SQLite.

---

## Features

✔ RFID Instrument Tracking

✔ Instrument Inventory

✔ Sterilization Status

✔ Operating Theatre Management

✔ Instrument History

✔ Reports Dashboard

✔ Search Instruments

✔ Admin Login
